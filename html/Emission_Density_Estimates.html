Agricultural NH3 emission densities

Description

The main sources of agricultural NH3 emissions surrounding each site are estimated and results calculated separately by source sector, where possible. The high-resolution holding-level agricultural statistics are subject to strict data licencing restrictions, therefore emissions from agricultural sectors associated with fewer than 5 agricultural holdings need to be merged with other emission sources to comply with data agreements in place, until non-disclosive outputs can be achieved. The data are summarised to show the relative contribution of the following source types; dairy cattle, other cattle, sheep, pigs, poultry, grass, crops and other sources. Where source sectors were disclosive (e.g. insufficient number of holdings with dairy cattle present), these were amalgamated into ‘other sources’ (containing a minimum of 5 holdings).

Important Notes

Although the finalised outputs of this script comply with the data licensing agreements of the agricultural census data, some of the intermediate objects (such as outputs from agric_ems_in_sf and holds_buff) contain sensitive information, which should not be saved!

Tabulated agricultural emission density estimates for Fenn’s, Whixall, Bettisfield, Wem and Cadney Mosses SAC

The example below demonstrates how to produce an agricultural emission density estimate for a designated site in the UK.

The example is for Fenn’s, Whixall, Bettisfield, Wem and Cadney Mosses SAC, which lies on the border of England and Wales. This site is frequently of interest for the development of Shared Nitrogen Action Plans as it situated in an intensive agricultural region.

In this example, farms within 2 km of the SAC (on both sides of the border) are include in the emission density estimate.

## Load packages

require(SpatialUKCEH)
require(AgricInvUKCEH)
require(sf)
require(pool)
require(stringr)
require(data.table)

# Load polygon to calculate the surrounding agricultural emission density for
sf_ste <- getSAC(id = "UK0012912")[1,]

# Create a bounding box surrounding the site
bbx <- st_bbox(sf_ste)

## DISCLOSIVE OUTPUT
# return all emissions on farms within 2km of the designated site
em <- agric_ems_in_sf(bbx, buffer = 2000, year = 2021, as_dt = T)

#
holds_buff <- agric_holds_in_buffer(sf_ste,
                                    buffer = 2000,
                                    mask = "UK",
                                    em = em)

agric_nh3_dens <- agric_dens_estimate(em,holds_buff,simplify = F)

The output contains the following fields:

  • BUFFER: The width of the buffer zone surrounding the site (in metres)
  • VARIABLE: The agricultural sector contributing to emissions
  • SECT_PCT: Percentage (%) value the sector contributes to agricultural NH3 emissions within the buffer zone
  • KG_NH3_HA: The emission density estimate for each sector in kg NH3 ha-1 year-1
  • N_HOLDS: The number of holdings contributing to

Create a plot of agricultural emissions within 10km of designated site

The example below produces agricultural emission density estimates for consecutive buffers (1 - 10 km from site boundary) for Fenn’s, Whixall, Bettisfield, Wem and Cadney Mosses SAC. The results are then visualised as a stacked bar chart, which we typically include in reports.

## Load packages

require(SpatialUKCEH)
require(AgricInvUKCEH)
require(sf)
require(pool)
require(stringr)
require(data.table)
require(ggplot2)
require(stringr)

# Load polygon to calculate the surrounding agricultural emission density for
sf_ste <- getSAC(id = "UK0012912")[1,]

# Create a bounding box surrounding the site
bbx <- st_bbox(sf_ste)

## loop through buffer zones 1 - 10km
dt <- lapply(X = seq(1000,10000,1000), FUN = function(buffer){
  
em <- agric_ems_in_sf(bbx, buffer = buffer, year = 2021, as_dt = T)

#
holds_buff <- agric_holds_in_buffer(sf_ste,
                                    buffer = buffer,
                                    mask = "UK",
                                    em = em)

agric_dens_estimate(em,holds_buff,simplify = F)

}) %>% do.call(rbind,.)

txt <- c("Dairy","Beef","Sheep","Pig","Poultry",
         "Minorlivestock","Grass","Crop","Other Sources*")

labs <- c("Dairy","Beef","Sheep","Pigs","Poultry",
          "Horses, Goats & Deer","Mineral Fertiliser",
          "Mineral Fertiliser","Other agircultural sources")

cols <- c("#ba2e51","#454bf7","#9e94a3","#ff96f3",
          "#ffe900","#877d61","#5a9152","#44caf7")

dt[,fact := factor(str_to_title(VARIABLE),txt,labs)]

p <- ggplot(data = dt, 
            aes(x = BUFFER, y = KG_NH3_HA, fill = fact))

p <- p + geom_bar(stat = "identity",colour=NA)



p <- p + scale_y_continuous(expand = expansion( mult = c(0,0.2) ),
                            name = bquote(atop("Total agricultural ammonia emission density",
                          "("~kg~NH[3]~ha^-1~yr^-1~")")))

p <- p + scale_x_continuous(expand = c(0,0), breaks = seq(2000,10000,2000),
                            labels = paste0("0 - ",seq(2,10,2)," km"),
